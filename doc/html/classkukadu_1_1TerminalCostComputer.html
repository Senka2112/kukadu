Detailed Description

The TerminalCostComputer implements the CostComputer interface.

This class implements the CostComputer in a simple way. The cost of the last rollout is inserted manually by the user to the console. This method can be used for very low dimensional reinforcement learning as there a low number of rollouts is needed.
